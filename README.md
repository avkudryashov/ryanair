# Поиск дешевых рейсов Ryanair из Валенсии

## Установка
```bash
pip install -r requirements.txt
```

## Использование
```bash
# Обычный поиск
python3 main.py -d 2026-02-15 -n 1,2,3

# Однодневные поездки с ночевкой (2 месяца вперед)
python3 main.py --one-day -e IBZ,PMI
```

Настройки в `config.yaml`: макс цена, исключенные страны, аэропорт отправления
